Factory Machine Monitoring System

A full-stack factory machine monitoring web application developed using React.js and Express.js to track machine status, monitor performance, and manage factory operations efficiently. The system provides a responsive dashboard for real-time monitoring and streamlined management of industrial machines.

🚀 Features
Real-time machine monitoring and status tracking
User-friendly dashboard with responsive UI
Machine performance and operational data management
RESTful API integration between frontend and backend
Secure backend architecture using Express.js
🛠️ Tech Stack

React.js
Tailwind CSS / Bootstrap
Axios
Backend
Node.js
Express.js

⚙️ Installation
Clone the repository
git clone https://github.com/shashank19p/Factory_machine_monitering.git
cd Factory_machine_monitering
Install dependencies
Frontend
cd frontend
npm install
Backend
cd backend
npm install
▶️ Run the Project
Start backend server
cd backend
npm start
Start frontend
cd frontend
npm run dev
🌐 Application Workflow
Factory machines are monitored through the dashboard
